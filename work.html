  • Measuring Internet fragmentation
  • Curricula for machine learning bias/fairness
  • Games for identifying (sociotechnical) security threats
  • Social aspects of biosensing
  • Speculative and future-oriented methodology
  • Brain-based authentication (or passthoughts)
  • Cognitive science, particularly spatial memory

I'm sure there are throughlines between these projects, but to be honest, I don't look for them. I do whatever interests me.
