I straddle academic research and critical technical practice (a la Phil Agre). This site is the second one. Lately, I'm interested in this internet's discontents: DAOs, paranets, meshes, alternative internets. You can follow that work on my substack.

Some things I've worked on in the past, in roughly reverse-chronological order:

  • Structural risks to the global Internet
  • Internet fragmentation
  • Curricula for machine learning bias/fairness
  • Games for identifying (sociotechnical) security threats
  • Can machines read the mind ? Do people think machines can read the mind? Do people act like they think machines can read the mind ?
  • Speculative and future-oriented methodology
  • Brain-based authentication (or passthoughts)
  • Cognitive science, particularly spatial memory

I'm sure there are throughlines between these projects, but to be honest, I haven't looked for them yet. I'm like a dog with a toy.
